# EDA---Zomato-Bangalore-Restaurants
- Exploratory Data Analysis on Dataset Zomato Bangalore Restaurants -**Data Walkthrough, Data Cleaning , Data Visualization** For **Key Insights**.
# Overview
- This Project Performs an End to End Exploratory Data Analysis on **Zomato Bangalore Restaurants** Dataset From **Kaggle** with **51717 Rows & 17 columns**
- The Goal was to Uncover the Key Hidden Insights Using Exploratory Data Analysis

# Key Findings
- North Indian cuisine ranks #1 in Bangalore even in a South Indian city, it dominates.
- Beverage shops and sweet shops offer the lowest average cost for two (~₹250).
- Online order availability does not significantly affect ratings.
- BTM has nearly twice as many restaurants as the second-most dense neighbourhood, HSR.
- Top-rated neighbourhoods include Lavelle Road, Koramangala 3rd Block, St. Marks Road, Koramangala 5th Block, and Church Street.
- Votes and ratings show a positive correlation.
- Only ~12.5% of restaurants offer online table booking.
- Table booking availability does not correlate with higher cost for two people.
